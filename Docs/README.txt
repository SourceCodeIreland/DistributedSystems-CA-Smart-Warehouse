# Smart Warehouse Management System

The Smart Warehouse Management System is a software solution designed to streamline warehouse operations by automating inventory management, order processing, and shipment logistics. This system utilizes a microservices architecture and gRPC communication protocol for efficient communication between services.

## Features

- **Inventory Management:** Manage stock items, add new items, update existing items, and delete obsolete items.
- **Order Processing:** Place new orders, cancel existing orders, update order statuses, and retrieve order details.
- **Shipment Logistics:** Schedule shipments, track shipment statuses, and update shipment details.
